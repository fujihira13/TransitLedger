# Product Overview

交通費と交際費を**最短入力**で記録し、月次で把握できる個人向けPWAアプリケーション。

## Core Capabilities

1. **最短入力での支出記録** - 平均10秒以内、3〜6タップで保存完了を目標
2. **ローカルファーストのデータ管理** - オフライン動作、端末内保存（localStorage）
3. **即座の集計表示** - 今月の交通費/交際費/合計が即座に見える
4. **入力最適化機能** - テンプレート、前回値引き継ぎ、頻出金額候補
5. **PWA対応** - ホーム画面追加、オフライン利用可能

## Target Use Cases

- **個人の交通費・交際費管理** - ログイン不要、個人データのみ
- **スマホ優先の利用** - iOS Safari / Android Chromeでの利用を想定
- **継続的な記録** - 「入力が面倒で続かない」を避ける設計

## Value Proposition

**継続しやすい導線**を最優先。入力速度と操作性を重視し、データ入力の心理的負担を最小化する。オフライン動作により、いつでもどこでも記録可能。
